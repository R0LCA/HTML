Descripción del libro

Desde los inicios de Internet a nuestros días, la Web se ha convertido en una parte fundamental de nuestras vidas.

Este libro está diseñado para aquellos que desean aprender sobre Hypertext Markup Language (HTML) y Cascading Style Sheets (CSS).

Contenido del libro

A lo largo de este libro se explorarán temas como la creación de páginas Web, la estructura de documentos en HTML y el uso de CSS para mejorar la apariencia de los sitios.
